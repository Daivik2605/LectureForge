# Frontend

Next.js app for upload, processing, and results views.

Quick notes:
- Copy `frontend/.env.example` to `frontend/.env.local`.
- Start dev server from `frontend/`: `npm run dev`
- Ensure `NEXT_PUBLIC_API_URL` points at the FastAPI backend.
- Config files use CommonJS with `.cjs` extensions (`postcss.config.cjs`, `tailwind.config.cjs`).
